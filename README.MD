**Game Design Document (GDD)**
**ชื่อโปรเจกต์:** Hot Rush!!!
**แนวเกม:** 2D Arcade Running / Racing
**แพลตฟอร์ม:** PC / Mobile (ปรับแต่ง UI ตามเหมาะสม)
**ธีมเกม:** การแข่งขันวิ่งมาราธอนสุดเดือดในช่วงเทศกาลสงกรานต์ของไทย ที่อากาศร้อนจัดจนนักวิ่งต้องระวังอาการ Overheat!

---

### **1. ภาพรวมของเกม (Game Overview)**
ผู้เล่นจะได้รับบทเป็นนักวิ่งที่เข้าร่วมการแข่งขัน "Hot Rush" ในช่วงสงกรานต์ เป้าหมายคือการวิ่งเข้าเส้นชัยเป็นอันดับ 1 จากผู้เข้าแข่งขันทั้งหมด 5 คน (ผู้เล่น 1 + CPU 4) ตัวเกมใช้มุมมอง 2D (Side-scrolling หรือ Top-down perspective ที่เห็นแทร็กชัดเจน) ขับเคลื่อนด้วยระบบกลไกการปั๊มปุ่ม (Button Mashing) เพื่อทำความเร็ว พร้อมกับการจัดการเกจความร้อน (Heat Gauge) และการใช้ไอเทมก่อกวนคู่แข่ง

---

### **2. ระบบการเล่นหลัก (Core Mechanics)**

**2.1 ระบบการเคลื่อนที่และ Spline Track**
* **Spline Path:** เส้นทางการวิ่งจะถูกกำหนดไว้ด้วย Spline (เส้นโค้งของสนาม) ตัวละครจะวิ่งไปตามเส้นนี้อัตโนมัติเมื่อมีการกดทำความเร็ว
* **Track Offset (การเบี่ยงเลน):** ผู้เล่นสามารถกด ขึ้น-ลง (Up/Down) เพื่อขยับตัวละคร (Offset) ออกจากจุดกึ่งกลางของ Spline ได้ โดยจะขยับได้ไม่เกินความกว้างของขอบถนน (Track Width)
* **ประโยชน์ของการ Offset:** ใช้สำหรับหลบสิ่งกีดขวาง หลบการโจมตีจากไอเทมของคู่แข่ง และขยับไปเก็บกล่องสุ่มไอเทมที่วางกระจายอยู่ตามเลนต่างๆ

**2.2 ระบบวิ่งและความร้อน (Run & Overheat System)**
* **Run Button (ปุ่มวิ่ง):** ผู้เล่นต้องรัวปุ่มวิ่ง (Mash) เพื่อให้ตัวละครมีความเร็ว ยิ่งรัวเร็ว ตัวละครยิ่งวิ่งไว
* **Heat Gauge (เกจความร้อน):** ทุกครั้งที่ปั๊มปุ่มวิ่ง เกจความร้อนจะพุ่งสูงขึ้น (สะท้อนถึงอากาศที่ร้อนจัดในวันสงกรานต์และการออกแรง) หากเกจค่อยๆ ลดลงเองเมื่อหยุดรัวปุ่ม แต่ความเร็วก็จะตกตามไปด้วย
* **Overheat State (สภาวะโอเวอร์ฮีต):** หากผู้เล่นปั๊มปุ่มวิ่งมากเกินไปจนเกจเต็ม ตัวละครจะติดสถานะ Overheat (ตัวแดง หอบเหนื่อย วิ่งช้าลงมากหรือหยุดชะงัก)
* **Cooldown Button (ปุ่มคูลดาวน์):** เมื่อเกิด Overheat ผู้เล่นต้องหยุดรัวปุ่มวิ่ง และหันไปรัวปุ่ม Cooldown (สาดน้ำใส่ตัวเอง) อย่างรวดเร็วเพื่อให้เกจความร้อนลดลงมาอยู่ในจุดสมดุล (Balanced) เมื่อเกจลดลงมาอยู่ในระดับปกติ สภาวะ Overheat จะหายไปและสามารถกลับมาปั๊มปุ่มวิ่งต่อได้

---

### **3. ระบบผู้เข้าแข่งขัน (AI & Opponents)**
* **จำนวนผู้แข่ง:** ผู้เล่น 1 คน แข่งกับ CPU 4 ตัว (รวมเป็น 5 ตัวละครในสนาม)
* **AI Behavior:** CPU จะมีความสามารถในการปั๊มปุ่มที่แตกต่างกัน (บางตัววิ่งเร็วแต่ Overheat บ่อย, บางตัววิ่งช้าแต่มั่นคง) และ AI สามารถ Offset ขึ้นลงเพื่อเก็บกล่องไอเทมและสาดไอเทมใส่เราได้

---

### **4. ระบบไอเทม (Item System)**
ตลอดเส้นทาง (Spline) จะมี **"ขันน้ำวิเศษ"** (Item Box) วางอยู่ตามเลนต่างๆ เมื่อผู้เล่นหรือ CPU วิ่งชน จะเป็นการสุ่มไอเทม 1 ใน 6 แบบ เพื่อใช้ก่อกวนคนอื่น หรือช่วยเหลือตัวเอง ดังนี้:

1.  **ปืนฉีดน้ำแรงดันสูง (Water Gun Snipe):** ยิงกระสุนน้ำพุ่งไปข้างหน้าเป็นเส้นตรงตาม Spline หากโดนคู่แข่งจะทำให้คู่แข่งกระเด็นถอยหลังและเสียจังหวะ
2.  **ถังน้ำแข็ง (Ice Bucket):** *ไอเทมบัฟตัวเอง* เมื่อใช้จะรีเซ็ตเกจความร้อนของตัวเองให้เป็น 0 ทันที และป้องกันอาการ Overheat เป็นเวลา 3 วินาที (ทำให้ปั๊มปุ่มวิ่งได้เต็มที่แบบไม่ต้องกลัวฮีต)
3.  **แป้งดินสอพอง (Slippery Powder):** ปาแป้งไปข้างหน้า หรือวางดักไว้ข้างหลัง หากใครเหยียบจะลื่น บังคับทิศทาง (Offset ขึ้นลง) ไม่ได้ชั่วขณะ และจะไถลไปตามขอบแทร็ก
4.  **แดดเปรี้ยง (Sun Flare):** สาดแสงแดดใส่ CPU ที่อยู่ข้างหน้าอันดับเรา 1 ตำแหน่ง ทำให้เกจความร้อนของเป้าหมายเพิ่มขึ้นอย่างรวดเร็ว (บีบให้เป้าหมาย Overheat ง่ายขึ้น)
5.  **ลูกโป่งน้ำ (Water Balloon):** ล็อคเป้าหมายอัตโนมัติไปยังผู้ที่อยู่อันดับ 1 (หรือคนข้างหน้า) ปาลูกโป่งน้ำอัดหน้า ทำให้ความเร็วลดลงชั่วขณะ และมีรอยน้ำบังหน้าจอเล็กน้อย
6.  **น้ำอบชูกำลัง (Thai Perfume Dash):** พุ่งตัว (Dash) ไปข้างหน้าอย่างรวดเร็วเป็นระยะเวลาสั้นๆ โดยไม่เพิ่มเกจความร้อน และสามารถทะลวงสิ่งกีดขวางหรือไอเทมดักจับได้

---

### **5. การควบคุม (Controls)**
* **Action 1 (ปั๊มปุ่มวิ่ง):** ปุ่ม Spacebar / ปุ่ม A (บนจอย) / แตะหน้าจอรัวๆ ซีกขวา
* **Action 2 (ปั๊มปุ่ม Cooldown):** ปุ่ม Shift / ปุ่ม B (บนจอย) / แตะหน้าจอรัวๆ ซีกซ้าย (จะกดได้ผลดีที่สุดตอน Overheat)
* **Action 3 (ใช้งานไอเทม):** ปุ่ม E / ปุ่ม X (บนจอย) / ปุ่มกดใช้ไอเทมบนจอ
* **Movement (ขยับเลน):** ปุ่มลูกศร ขึ้น-ลง / D-Pad ขึ้น-ลง / ปัดหน้าจอ ขึ้น-ลง

---

### **6. การออกแบบศิลป์และเสียง (Art & Audio Direction)**
* **Art Style:** 2D ลายเส้นการ์ตูนสีสันสดใส (Vibrant Colors) แทร็กวิ่งอาจเป็นถนนในเมืองไทยที่มีคนสาดน้ำอยู่ข้างทาง มีรถตุ๊กตุ๊ก หรือเสาไฟเป็นพื้นหลัง
* **Animation:** ตัวละครตอนสับขาวิ่งต้องดูตลกและโอเวอร์ (Exaggerated) ตอน Overheat หน้าจะแดง มีควันออกหู คลานกระดึ๊บๆ ตอน Cooldown จะมีแอนิเมชันเอาน้ำราดหัวตัวเอง
* **Audio:** ดนตรีแนวสามช่าหรือ EDM สนุกสนานเร้าใจจังหวะเร็วๆ (BPM สูง) เสียงปั๊มปุ่มเป็นเสียงสับเท้า เสียงรัวปุ่ม Cooldown เป็นเสียงสาดน้ำ "สาดๆๆๆ" และมีเสียงนกหวีดหรือเสียงเฮของคนดูเมื่อเข้าเส้นชัย
